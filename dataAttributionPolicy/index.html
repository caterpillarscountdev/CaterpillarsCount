Data Attribution Policy

Caterpillars Count! recommends that all Caterpillars Count! data and/or other forms of information from the Caterpillars Count! Website downloaded by a user, referred to as "User," be acknowledged as received from Caterpillars Count! according to the following terms of this Caterpillars Count! Data Attribution Policy.
  1. The following acknowledgments should be included either in the main text or Acknowledgments section of all publications using data distributed by Caterpillars Count!: "Data were provided by Caterpillars Count! and the many participants who contribute to its foliage arthropod monitoring program."
  2. All data sets distributed by Caterpillars Count! should be cited as described below:
    • Caterpillars Count! [Year of dataset access]. Foliage Arthropod Data. Data type: [Raw Data, Map Data, Phenology Data, or Composition Data]. [Date range of data used] for Site: [Site Name]. Chapel Hill, North Carolina, USA. Data set accessed [Date] at https://caterpillarscount.unc.edu.
    • Examples:
      • Caterpillars Count!. . Foliage Arthropod Data. Data type: Phenology Data. 01/01/2017-12/31/2019 for Site: NC Botanical Garden. Chapel Hill, North Carolina, USA. Data set accessed at https://caterpillarscount.unc.edu.
      • Caterpillars Count!. . Foliage Arthropod Data. Data type: Raw Data. 01/01/2010-12/31/2019 for Site: All sites. Chapel Hill, North Carolina, USA. Data set accessed at https://caterpillarscount.unc.edu.
  3. We request that all bibliographic information be provided to Caterpillars Count! (caterpillarscount@gmail.com) for all manuscripts using Caterpillars Count! data immediately after their publication.
